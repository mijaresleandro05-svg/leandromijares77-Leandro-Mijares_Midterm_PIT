# Interactive Resume - Leandro Vercando Mijares

## Simple Interactive Resume Website

This is a clean, professional one-page resume with interactive features built with HTML, CSS, JavaScript, and jQuery.

## Features

**Interactive Elements:**
- ✓ Click "Read More" button to expand about section
- ✓ Form validation with error messages
- ✓ Skill bars animate on hover
- ✓ Smooth scrolling navigation
